The ``3-say_my_name`` module
============================

Using ``say_my_name``
---------------------

Importing the function:
>>> say_my_name = __import__('3-say_my_name').say_my_name

1. Standard Usage (Happy path)
==============================
Normal names:
>>> say_my_name("Walter", "White")
My name is Walter White

>>> say_my_name("Jesse", "Pinkman")
My name is Jesse Pinkman

Names with spaces (valid strings):
>>> say_my_name("Jean Claude", "Van Damme")
My name is Jean Claude Van Damme

2. Usage with Default Argument
==============================
Only first name provided:
>>> say_my_name("Heisenberg")
My name is Heisenberg 

>>> say_my_name("Tuco")
My name is Tuco 

3. Edge Cases: Empty or Weird Strings
=====================================
Note: Empty strings are still strings, so no error should be raised, just weird spacing in output.

Both empty:
>>> say_my_name("", "")
My name is  

First name empty:
>>> say_my_name("", "Schrader")
My name is  Schrader

Last name empty (explicitly passed):
>>> say_my_name("Hank", "")
My name is Hank 

Strings with only spaces:
>>> say_my_name("   ", "   ")
My name is
