The `3-say_my_name` module
==========================

>>> say_my_name = __import__('3-say_my_name').say_my_name

Basic usage
===========

>>> say_my_name("Jean", "Dujardin")
My name is Jean Dujardin

>>> say_my_name("Breaking", "Bad")
My name is Breaking Bad

Default last_name
=================

>>> say_my_name("Antho")
My name is Antho
